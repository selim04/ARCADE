When you highlight a name in the column on the left and click on choose, a new window will open offering you the following options:

This allows you to call an individual or group of up to 10 participants using skype
If the person chosen is not in your list of skype contacts, you can add them
You can begin a text chat within skype
You can send a document, picture or other file to another participant using skype
The person chosen must already subscribe to this service within skype if you want to send a voicemail
If you don't already have skype this will allow you to set up an account and download the application to your computer. If you already have an account but skype isn't open on your computer this will open it
